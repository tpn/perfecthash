# Release Guide

This directory contains helper scripts to build, test, install, package, and
publish PerfectHash release artifacts.

## Core Principles

- Release versions come from git tags (`vX.Y.Z`) by default.
- No post-tag CMake version bump is required.
- `tag + push` is the default release trigger path.
- Build profile is explicit (`full`, `online-rawdog`, `online-rawdog-llvm`).

## Release Scripts

- Cut tag helper: `ci/cut-release.sh`
- Linux: `ci/release-linux.sh`
- macOS: `ci/release-macos.sh`
- Windows: `ci/release-windows.ps1`
- Shared helpers: `ci/release-common.sh`
- Notes generation: `ci/generate-release-notes.sh`

## Cutting A Release Tag

Create a local tag plus generated notes:

```bash
ci/cut-release.sh --version 0.63.0
```

Create and push in one step:

```bash
ci/cut-release.sh --version 0.63.0 --push
```

This helper performs preflight checks:

- Valid version format.
- Clean working tree (unless `--allow-dirty`).
- No local/remote tag collisions.
- Release notes generation via `ci/generate-release-notes.sh`.

## Version Resolution Order

All release scripts resolve version in this order:

1. explicit `--version` / `-ReleaseVersion` or `RELEASE_VERSION`
2. exact git tag on `HEAD` matching `v*`
3. latest git tag matching `v*`
4. `CMakeLists.txt` fallback

## Build Profiles

- `full`: full build including CLI executables and online/JIT targets.
- `online-rawdog`: slim online/rawdog package.
- `online-rawdog-llvm`: online/rawdog + LLVM support.

Artifacts include profile in the filename, for example:

- `perfecthash-0.63.0-full-linux-x86_64.tar.gz`
- `perfecthash-0.63.0-online-rawdog-macos-arm64.tar.gz`

## GitHub Actions Release

`release.yml` behavior:

- Tag pushes (`v*`) build Linux/macOS/Windows artifacts and publish a GitHub release.
- Workflow dispatch can run profile-specific builds without publishing.
- Release body is generated by `ci/generate-release-notes.sh` and can include
  curated content from `RELEASE-NOTES.md`.

## Local Release Examples

Linux:

```bash
ci/release-linux.sh --profile full --clean
```

macOS:

```bash
ci/release-macos.sh --profile online-rawdog --arch arm64 --clean
```

Windows:

```powershell
pwsh -File ci/release-windows.ps1 -BuildProfile full -Clean
```
